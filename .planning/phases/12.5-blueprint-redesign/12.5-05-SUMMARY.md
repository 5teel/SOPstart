## Online Voice Capture Architecture

The Wave 4 SUMMARY documented the online blob upload as a known stub. This plan closes it:

1. `VoiceCaptureControl.stop()` returns `{ blob, transcript, confidence, ext }`
2. `onTranscript(transcript, confidence)` fires immediately — UI shows the transcript
3. `onCaptured({ blob, transcript, confidence, ext })` fires next (online path only, when transcript is non-empty)
4. Block's `handleCaptured` decodes JWT to extract `organisation_id`, constructs `{orgId}/voice/{sopId}/{uuid}.{ext}` path, uploads to sop-voice-notes bucket, calls `saveVoiceNote` with the storage path
5. Upload failure is silent (console.error only) — transcript already visible to worker

## Flow Graph Derivation

`deriveFlowGraph(sop)` produces a top-down single-column layout:
- Each step across all sections becomes a `'step'` node (or a typed node if the parent section's `layout_data.content[]` has a matching block type)
- Positions: `{ x: 0, y: i * 100 }` — 100px vertical gap between nodes
- Sequential edges connect consecutive nodes
- Node type detection: checks `section.layout_data.content[].type` for MeasurementBlock, DecisionBlock, EscalateBlock, SignOffBlock, InspectBlock, ZoneBlock
