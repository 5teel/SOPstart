  <name>Task 1: Install Playwright and create test configuration</name>
  <files>
    package.json, playwright.config.ts
  </files>
  <action>
    1. Install Playwright as a dev dependency:
       ```
       npm install -D @playwright/test
       npx playwright install --with-deps chromium
       ```
       Only install Chromium browser — we do not need Firefox/WebKit for integration tests.

    2. Create `playwright.config.ts` at the project root:
       ```typescript
       import { defineConfig } from '@playwright/test'

       export default defineConfig({
         testDir: './tests',
         testMatch: '**/*.test.ts',
         timeout: 30_000,
         retries: 0,
         use: {
           baseURL: 'http://localhost:3000',
         },
         projects: [
           {
             name: 'integration',
             testMatch: /rls-isolation|auth-flows/,
           },
           {
             name: 'e2e',
             testMatch: /offline-indicator/,
             use: { browserName: 'chromium' },
           },
         ],
       })
       ```

    3. Add test scripts to `package.json`:
       ```json
       "scripts": {
         "test": "npx playwright test",
         "test:integration": "npx playwright test --project=integration",
         "test:e2e": "npx playwright test --project=e2e"
       }
       ```

    4. Add `test-results/` and `playwright-report/` to `.gitignore`.

    NOTE: Do NOT install vitest. RESEARCH.md specifies Playwright as the test tool for this phase. If vitest was listed as a dev dependency in Plan 01-01 Task 1 step 3, the executor of Plan 01-01 should replace `vitest @vitest/coverage-v8` with `@playwright/test` in that install command (Plan 01-01 has been updated to reflect this).
